# Image Processing Toolkit

Этот проект представляет собой консольное приложение для выполнения задач по обработке изображений. Он содержит две реализации ключевых алгоритмов:

1.  **`library`**: Быстрая реализация, использующая высокоуровневые функции библиотеки **OpenCV**.
2.  **`custom`**: Пользовательская реализация, написанная с нуля с использованием **NumPy** для образовательных целей и демонстрации алгоритмов.

Приложение позволяет сравнивать результаты и производительность обеих реализаций.

## Реализованные функции

-   **Обнаружение границ**:
    -   `custom`: Оператор Собеля.
    -   `library`: Алгоритм Кэнни.
-   **Обнаружение углов**:
    -   `custom` и `library`: Алгоритм Харриса.
-   **Измерение производительности**: Вывод времени выполнения для каждой операции.

## Структура проекта

-   `main.py`: Точка входа и интерфейс командной строки (CLI).
-   `/interfaces`: Содержит абстрактный базовый класс `IImageProcessing`, определяющий контракт для всех реализаций.
-   `/implementation`: Содержит конкретные реализации:
    -   `image_processing.py`: Реализация на основе OpenCV.
    -   `image_processing_custom.py`: Реализация на основе NumPy.
-   `/test_images`: Папка с изображениями для тестирования.

## Использование

Программа запускается из командной строки со следующими аргументами:

```
python main.py <метод> <путь_к_изображению> [--impl <реализация>] [-o <путь_для_сохранения>]
```

### Аргументы

-   `<метод>` (обязательный): `edges` или `corners`.
-   `<путь_к_изображению>` (обязательный): Путь к входному файлу.
-   `--impl` (необязательный): `custom` (по умолчанию) или `library`.
-   `-o` (необязательный): Путь для сохранения файла результата. Если не указан, имя генерируется автоматически.

### Примеры команд

#### 1. Обнаружение границ (ваша реализация)
```bash
python main.py edges ./test_images/billiard.png --impl custom
```
*(Результат будет сохранен в `billiard_edges_custom.png`)*

#### 2. Обнаружение границ (библиотечная реализация)
```bash
python main.py edges ./test_images/billiard.png --impl library
```
*(Результат будет сохранен в `billiard_edges_library.png`)*

#### 3. Обнаружение углов с указанием пути сохранения
```bash
python main.py corners ./test_images/NotreDame.jpg --impl custom -o ./my_corners.jpg```
*(Результат будет сохранен в `my_corners.jpg`)*
